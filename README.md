# SIMAS
## Student Academic Monitoring Information System

SIMAS (Student Academic Monitoring Information System) is a desktop-based application developed to monitor and manage students' academic data in schools. The system helps administrators and teachers manage student information, attendance, and academic performance efficiently.

## 📌 Features

- Student data management
- Teacher data management
- Academic monitoring
- Attendance recording
- Grade management
- Academic report generation
- Dashboard for monitoring student performance
